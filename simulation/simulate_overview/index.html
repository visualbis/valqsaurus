Simulate Tab

Visualize the model you have built in a tree/table format on ValQ's Simulate tab.

Highlights of Simulate tab:

  • Simulate on-the-fly and perform what-if analysis
  • Create and compare multiple scenarios
  • Assign important nodes as KPIs and key inputs to assess progress
  • Evaluate performance against a benchmark
  • Comment & Collaborate with your team
  • Save to Microsoft Azure

Check out the Simulate tab below:

The left side panel is called the Navigation Panel and on the right is the ValQ canvas. The Simulate tab on ValQ has it's own properties and here's the discussion on Plan tab Vs Simulate tab for better understanding.

Pre-requisite#

A ValQ model is required before you can start using the Simulate tab.

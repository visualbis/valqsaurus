Table View

There are two ways to switch to the Table View in the Simulate screen:

  1. By clicking on the table icon at the top of the Simulate tab canvas.
  2. By clicking on the Table option under the Visualization section of the Navigation Panel.

Here's the Table View on full-screen mode after collapsing the Navigation Panel.

Features of Table View#

The features of the table view discussed are as follows:

  1. Search for a node
  2. Perform simulation
  3. Navigation Panel
  4. Node details Pop-up
  5. Go to Plan
  6. Add Comments
  7. Summarized view
  8. Period view
  9. Customize columns
  10. Size columns to fit

1. Search for a node#

Type the node you want to search and select the same from the dropdown.


The selected node and its descendants will be displayed.

2. Perform simulation#

Hover over the trend line on a node to perform simulation. The simulation results for each node with percentage change are available on top of the canvas.


Click on the Edit icon to manually change the values.


Use the bin icon to remove a simulation and remove all the simulation at once, click on Reset all option.

3. Navigation Panel#

Below is the Table View with the Navigation Panel. The Navigation Panel features for the Tree View applies here.

4. Node details Pop-up#

Click on any node title to view its node details.

5. Go to Plan#

The plan icon takes you to the recent access plan and displays that particular node, its siblings and children. This feature is same as the Go to Plan with Tree View.

Plan icon next to Production Cost node

Plan series

6. Add Comments#

Click on the comment icon to open the Comments section. Comments can be added/edited/deleted to the model at node granularity. This feature is same Add Comments with Tree View.

7. Summarized view#

By default, the table is shown in the summarized mode.

  • Before performing simulations - aggregated baseline and comparison series will be displayed.

  • After performing simulations - original and simulated baseline, comparison values and simulation impact will be displayed.

8. Period view#


On selecting the period view from the settings, you'll be able to view the table by periods.

9. Customize columns#

To customize the columns in the table view, select the Settings icon at the top right corner, and then select Settings option. On the left, search/select or unselect the columns that you want to display. On the right, you can reorder columns by drag and drop.

Manage table columns

To default to the settings in the Table columns section, select Reset to default from the bottom of the window.

Select Apply to save your changes.

10. Size Columns to fit#

Use the Size columns to fit option to auto-size the columns to fit the screen width for a better view.

## Introduction

### What is ValQ

ValQ is a serverless, lightweight, multi-purpose application running on Power BI supporting use cases such as planning, forecasting, budgeting, time series forecasting, and value driver planning. It helps enterprises run simulations, plans & analyses in a very visual and intuitive manner.

### Supported platforms

### System requirements

Following are the recommended system requirements.

1. Operating System - Windows Server 2012 or later
2. Processor - 2.4 GHz Quadcore or with similar CPU Performance
3. RAM- 16 GB

### Browser compatibility

1. Edge
2. Google Chrome
3. Mozilla Firefox
4. IE 11

### Databases

1. Microsoft SQL Server - Version 2016 or latest
2. MySQL Server - 5.6 or latest
3. Maria DB - 5.5 or latest

Database can be also installed in the Virtual Machine where we are installing ValQ Server.

### 4. Generate SSL Certificate

ValQ Private tenant requires an SSL Certificate for proper working and secure communication. 

There are 2 ways to generate an SSL Certificate:

4.1 Buy or generate an SSL certificate for the ValQ private tenant VM in the x.509 if you have a vendor. 

4.2 Generate self signed certificate by using the *Open SSL tool:*

- Open Windows File Explorer
- Navigate to the OpenSSL bin directory, generally it is `c:\OpenSSL\bin\`
- Right-click the **openssl.exe** file and select **Run as administrator**
- Enter the following command to begin generating a certificate and private key:

```bash
**req -x509 -sha256 -nodes -days 365 -newkey rsa:2048 -keyout privateKey.key -out certificate.crt**
```

- Enter a 7-digit **applicable Distinguished Name (DN)**
- Once completed, you will find the **certificate.crt** and **privateKey.key** files in the same bin directory location

## Certificate Configuration

1.Update **Nginx configuration files**

- Go to the **ValQ installation directory** on your machine
- Open  **nginx > confs** folder
- Replace the **Nginx Server** existing *Security Certificate and KEY files with the [SSL Certificates](https://www.notion.so/ValQ-Private-Tenant-Installation-Guide-9525981b1f0b407fa7cad823856e3f04#05752a892828439e8f368ab40cf841de) keys*
    - The new SSL certificate keys must be of the *same name and in the same format as the original server keys*
    - *The file should not contain Pass Phrase.*
    - You can also generate the certificate by using the server.key by using Open SSL.
    - If you are using self signed certificate, then it should be made trusted in all the computers across the organization.

## Troubleshooting steps

## Un-installation

### Uninstall ValQ

1.Go to the **ValQ installation path** on your machine

2.Double click on the **Uninstall ValQ Collaborate** application

![ValQ%20Private%20Tenant%20Installation%20Guide%2010daae5753b34fe09e4be742ee79a33a/Uninstall_valQ_installer.png](ValQ%20Private%20Tenant%20Installation%20Guide%2010daae5753b34fe09e4be742ee79a33a/Uninstall_valQ_installer.png)

Uninstall_valQ_installer

3.Click **Yes** in the **User Account Control** popup window

![ValQ%20Private%20Tenant%20Installation%20Guide%2010daae5753b34fe09e4be742ee79a33a/Uninstall_yes.png](ValQ%20Private%20Tenant%20Installation%20Guide%2010daae5753b34fe09e4be742ee79a33a/Uninstall_yes.png)

Uninstall_yes

4.Click **Yes** to confirm the deletion of **ValQ Collaborate**

![ValQ%20Private%20Tenant%20Installation%20Guide%2010daae5753b34fe09e4be742ee79a33a/Uninstall_confirm.png](ValQ%20Private%20Tenant%20Installation%20Guide%2010daae5753b34fe09e4be742ee79a33a/Uninstall_confirm.png)

Uninstall_confirm

5.Uninstallation **Setup** begins and may take approximately 3.5 minutes to complete

![ValQ%20Private%20Tenant%20Installation%20Guide%2010daae5753b34fe09e4be742ee79a33a/Uninstalling-Setup_progress.png](ValQ%20Private%20Tenant%20Installation%20Guide%2010daae5753b34fe09e4be742ee79a33a/Uninstalling-Setup_progress.png)

Uninstalling-Setup_progress.png

6.Click **OK** to finish uninstalling **ValQ Collaborate**

![ValQ%20Private%20Tenant%20Installation%20Guide%2010daae5753b34fe09e4be742ee79a33a/Uninstallation_complete.png](ValQ%20Private%20Tenant%20Installation%20Guide%2010daae5753b34fe09e4be742ee79a33a/Uninstallation_complete.png)

Uninstallation_complete

7.Restart the System.

### Post-uninstallation step

Delete the residual files

- Go to **ValQ installation path**
- Delete all the remaining files present in the folder.

![ValQ%20Private%20Tenant%20Installation%20Guide%2010daae5753b34fe09e4be742ee79a33a/Delete_residual_files.png](ValQ%20Private%20Tenant%20Installation%20Guide%2010daae5753b34fe09e4be742ee79a33a/Delete_residual_files.png)

Delete residual files

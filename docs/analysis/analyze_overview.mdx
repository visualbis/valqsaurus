*Analyze tab and the 3 types of analysis*

Below are the 3 types of analysis that you can perform:

1. <Link to={useBaseUrl("analysis/variance_analysis")}>
     Variance Analysis
   </Link>: Compare multiple series/scenario to calculate the absolute and percentage variance for each node in the model.
1. <Link to={useBaseUrl("analysis/attribution-analysis")}>
     Attribution Analysis
   </Link>: Evaluate the simulation impact of various input nodes and it's cumulative effect on the Key Performance Indicator.
1. <Link to={useBaseUrl('analysis/sensitivity-analysis')}>Sensitivity Analysis</Link>: Identify and analyze how the output node gets affected in response to deviation in a value driver represented by way of a tornado chart.

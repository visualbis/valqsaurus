In this section, you will:
 - Learn how to [install](#get-valq) ValQ on Power BI Desktop
 - [Launch](#launch-valq) ValQ

## Get ValQ

You can use any of the following three ways to get ValQ added to Microsoft Power BI:

<div style={{textAlign: 'justify'}} className="table-center">

|   | Click on any option to know more  |                                                                                                                                                                                                                                                                   
|---|---------------------|
| 1 | [Import a visual from a file](#1-import-a-visual-from-a-file)  |
| 2 | [Get more visuals - AppSource](#2-get-more-visuals---appsource)  |
| 3 | [Get more visuals - My organization](#3-get-more-visuals---my-organization) |

</div>

### 1. Import a visual from a file

<div style={{textAlign: 'justify'}}>
Import a visual from a file option is used when you have ValQ as a .pbiviz file in your local machine. <br /> <br />
1. a) To get started, click on the ellipses under the Visualizations pane and choose Import a visual from a file option
as shown in the image below.
</div> <br />

<div style={{textAlign: 'center'}}>
  <Zoom>
<img alt="ValQ get" src={useBaseUrl('/doc-images/install/install2.png')}/>
  </Zoom>
</div> <br />

1. b) Now select the .pbiviz file available in your local desktop to import ValQ into Power BI. 
<br />

<div style={{textAlign: 'center'}}>
  <Zoom>
<img alt="ValQ get" src={useBaseUrl('/doc-images/install/install5.png')} />
  </Zoom>
</div> <br />

1. c) Once the file gets imported, you will see the message as shown below. Click OK.
<br />

<div style={{textAlign: 'center'}}>
  <Zoom>
<img alt="ValQ get" src={useBaseUrl('/doc-images/install/install6.png')} />
  </Zoom>
</div> <br />

1. d) You have successfully added ValQ to Microsoft Power BI. You will now be able to see ValQ now under the Visualizations pane. 
<br />

<div style={{textAlign: 'center'}}>
  <Zoom>
<img alt="ValQ get" src={useBaseUrl('/doc-images/install/install7.png')} />
  </Zoom>
</div>

### 2. Get more visuals - AppSource

Get more visuals option can be used to download ValQ from the AppSource marketplace.

<div style={{textAlign: 'justify'}}>
2. a) To get started, click on the ellipses under the Visualizations pane and choose Get more visuals option.
</div> <br />

<div style={{textAlign: 'center'}}>
  <Zoom>
<img alt="ValQ get" src={useBaseUrl('/doc-images/install/install3.png')}/>
  </Zoom>
</div> <br />

2. b) If you have not signed in to Microsoft Power BI already, you would be prompted to sign in to access the marketplace.
<br />

<div style={{textAlign: 'center'}}>
  <Zoom>
<img alt="ValQ get" src={useBaseUrl('/doc-images/install/install8.png')} />
  </Zoom>
</div> <br />

2. c) Simply search for ValQ in the search bar and add the visual to Power BI by clicking Add button.
<br />

<div style={{textAlign: 'center'}}>
  <Zoom>
<img alt="ValQ get" src={useBaseUrl('/doc-images/install/install9.png')} />
  </Zoom>
</div> <br />

2. d) Once the file gets imported, you will see the message as shown below. Click OK.
<br />

<div style={{textAlign: 'center'}}>
  <Zoom>
<img alt="ValQ get" src={useBaseUrl('/doc-images/install/install6.png')} />
  </Zoom>
</div> <br />

2. e) You have successfully added ValQ to Microsoft Power BI. You will now be able to see ValQ now under the Visualizations pane. 
<br />

<div style={{textAlign: 'center'}}>
  <Zoom>
<img alt="ValQ get" src={useBaseUrl('/doc-images/install/install4.png')} />
  </Zoom>
</div>

### 3. Get more visuals - My organization

Get more visuals option can also be used to download ValQ when the visual is a part of My organization marketplace.

<div style={{textAlign: 'justify'}}>
3. a) To get started, click on the ellipses under the Visualizations pane and choose Get more visuals option.
</div> <br />

<div style={{textAlign: 'center'}}>
  <Zoom>
<img alt="ValQ get" src={useBaseUrl('/doc-images/install/install3.png')}/>
  </Zoom>
</div> <br />

3. b) If you have not signed in to Microsoft Power BI already, you would be prompted to sign in to access the marketplace.
<br />

<div style={{textAlign: 'center'}}>
  <Zoom>
<img alt="ValQ get" src={useBaseUrl('/doc-images/install/install8.png')} />
  </Zoom>
</div> <br />

3. c) Switch to My organization tab to identify the ValQ visual and click Add to get ValQ. <br /> 

<div style={{textAlign: 'center'}}>
  <Zoom>
<img alt="ValQ get" src={useBaseUrl('/doc-images/install/install12.png')} />
  </Zoom>
</div> <br />

> Note <br />
 ValQ will be available here only if the administrator from your organization has enabled ValQ
<br />

3. d) Once the file gets imported, you will see the message as shown below. Click OK.
<br />

<div style={{textAlign: 'center'}}>
  <Zoom>
<img alt="ValQ get" src={useBaseUrl('/doc-images/install/install6.png')} />
  </Zoom>
</div><br />

3. e) You have successfully added ValQ to Microsoft Power BI. You will now be able to see ValQ now under the Visualizations pane. 
<br />

<div style={{textAlign: 'center'}}>
  <Zoom>
<img alt="ValQ get" src={useBaseUrl('/doc-images/install/install4.png')} />
  </Zoom>
</div>

- To begin, click on the ValQ icon. You will see ValQ visual on the canvas.
  <Zoom>
  <img alt="ValQ get" src={useBaseUrl('/doc-images/install/install13.png')} />
    </Zoom>

- Expand the visual to fit the screen. You can now start using ValQ by assigning value to the visual.
  <Zoom>
  <img alt="ValQ get" src={useBaseUrl('/doc-images/install/install14.png')} />
    </Zoom>

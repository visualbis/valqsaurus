## 1. Node components

First, let's take a look at what makes up a node

<div style={{textAlign: 'center'}}>
  <Zoom>
    <img alt="ValQ users" src={useBaseUrl('/doc-images/intro/node4.png')} />
  </Zoom>    
</div>
<br />

Each node represents a value driver, KPI or category and it's several performance metrics. Here's how to interpret a node:

<div style={{textAlign: 'justify'}} className="table-center">

|   |   Component	| Description  	|
|---    |---	|---	|
|1| Node name  	| Name of the value driver or KPI	|
|2| Conditional Formatting  	| Color band indicating the variance 	|
|3| Value Trend 	| A sparkline graph showing the trend |
|4| Baseline Value  	| Node value of the baseline series |
|5| Variance for all periods  	| Full year variance of the node |
|6| Variance for current period  	| Current month (usually first period in the series) month variance of the node |
|7| Absolute Variance  | Variance showed in absolute value |
|8| Percentage Variance  	| Variance showed in percentage |

</div> <br />

> **Note** <br />
Variance is calculated as the difference between the Baseline series and Comparison series

<div style={{textAlign: 'justify'}}>
Baseline series is the primary series and Comparison series is the benchmark against which the Baseline is compared.
Here the Sales Budget is the Baseline series and Sales Forecast is the Comparison series. 
</div> <br />

### 2.1 Tree view

<div style={{textAlign: 'justify'}}>
The tree structure is presented in a  horizontal format as a visual.  Just like a tree,
ValQ model has a stem on Top called Top node and the Top node then branches out to other nodes. 
</div> <br />

<div style={{textAlign: 'center'}}>
  <Zoom>
    <img alt="ValQ users" src={useBaseUrl('/doc-images/intro/node7.png')} />
  </Zoom>    
</div>
<br />

<div style={{textAlign: 'justify'}}>
The model is segmented into levels starting from level 1 with Top node, here it's Overall Result. 
The nodes share a parent-child relationship at each level.
For example, the East node is the Parent node while Consumer node is a Child node, Consumer node is the Parent node 
while Chris Morris node is a Child node. 
</div> <br />

<div style={{textAlign: 'center'}}>
  <Zoom>

### 1. Scenarios

Scenarios are particularly useful when you want to perform a what-if analysis to the model and you can save each instance as an individual scenario. This way you have multiple scenarios to analyze and conduct scenario analysis.

<div style={{ textAlign: "center" }}>
  <Zoom>
    <img
      alt="Scenarios"
      src={useBaseUrl("/doc-images/simulate_tab/scenarios-options.png")}
    />
  </Zoom>
</div>

The scenario with the yellow tag is the active scenario. 
- To collapse the scenario property click on the ( **^** ) option. 
- To create a new scenario, click on the ( **+** ) option.
- To download the selected scenario data as a Microsoft Excel-compatible XML file, click the ( <img width="15" src={useBaseUrl('img/download.svg')}/> ) icon.
- Click on the 3 dots `...` to access further options:
  - **Copy Scenario**: Duplicates the existing scenario into a new one.
  - **Edit**: Change the title of the scenario and add an optional description.
  - **Share**: Enable to share the scenario across users/groups if the model is shared.
  - **Delete**: Deletes a scenario.
- **Compare Scenarios** option helps you compare multiple scenarios:
   - Click **Manage Scenario** to add or remove scenarios for comparison.
   - To  download the scenario comparison as an excel file, click **Download to Excel**.

<div class="center">
  <Zoom>
    <img
      alt="Scenarios comparison"
      src={useBaseUrl("/doc-images/simulate_tab/compare-scenarios.png")}
    />
  </Zoom>
  <p>Scenarios comparison screen</p>
</div>

> **Note** <br /> Scenarios are not stored as Data Series on ValQ. Click <Link to={useBaseUrl('planning/plan_overview#1base-data-series-vs-scenario-data')}>here</Link> to know more on this.

> To rename the property's name to another value, go to the <Link to={useBaseUrl('model/settings#11-scenarios')}>General > Settings > Navigation Panel</Link> section.

### 3. Simulation Period

<div style={{textAlign: 'justify'}}>

ValQ model updates the simulated values based on the periods selected in the Simulation Period. You can select just one period or
choose a range of period with '[to]' icon, ValQ applies simulation and fetches simulated values only to those selected periods.
To change the period labels, go to <Link to={useBaseUrl('model/settings#13-simulation-period')}>settings</Link>.
Click on the **^** option to collapse this property.

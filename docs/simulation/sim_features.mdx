In this section we'll take a look at the Simulate tab and it's features. 

## 1. Scenarios

<div style={{textAlign: 'justify'}}>
Scenarios are particularly useful when you want to perform what-if analysis to the model and you can save each instance as an 
individual scenario. This way you have multiple scenarios to analyze and conduct scenario analysis. 
</div> <br />

<div style={{textAlign: 'center'}}>
  <Zoom>
<img alt="ValQ logo" src={useBaseUrl('/doc-images/Simulation/sim4.png')} />
  </Zoom>
</div> <br />

<div style={{textAlign: 'justify'}}>
The scenario with the yellow tag is the active scenario. To collapse the sceanrio property click on '^' option 
and to add a new scenario, click on '+'. Clicking on three dots gives you options to Copy Scenario, Edit and 
Delete. 
<br /> <br />

Copy Scenario - Creates a duplicate of the scenario that you have copied <br />
Edit - Change the title of the scenario and add description which is optional <br />
Delete - Deletes a scenario <br />

The URL icon is availablet to help you compare multiple scenarios, observe the
summary of the Scenarios and to take a look at the KPIs (see below Figure). With Manage Scenrios option 
you can add or remove scenarios for comparison and you also have the option to download the 
scenario comparison as an excel file while using ValQ on Power BI Service.  
</div> <br />

<div style={{textAlign: 'center'}}>
  <Zoom>

  </Zoom>

## 2. Sub Models

<div style={{textAlign: 'justify'}}>
Sub Models are part of the model and a node is added as a sub model to acccess it quickly. This property is useful 
when there are large number of nodes and you want to get to the important nodes on click of a button. You can go to <Link to={useBaseUrl('docs/model/settings#12-sub-models')}> settings </Link> to add or remove sub models on ValQ.
Click on '^' option to collapse this property.
</div> <br />

## 3. Simulation Period

<div style={{textAlign: 'justify'}}>
ValQ model updats the simulated values based on the periods selected in the Simulation Period. You can select just one period or
choose a range of period with '[to]' icon, ValQ applies simulation and fetches simulated values only to those selected periods.
To change the period labels, go to <Link to={useBaseUrl('docs/model/settings#13-simulation-period')}> settings </Link>. Click on '^' option to collapse this property.
</div> <br />

<div style={{textAlign: 'center'}}>

## 4. Value display

<div style={{textAlign: 'justify'}}>
Value Display makes available options of aggregation like month, year to date, full year and simulated periods and scaling options.
You can go to settings to change <Link to={useBaseUrl('docs/model/data-series#2-time-aggregation-labels')}> time aggregation labels </Link> and
<Link to={useBaseUrl('docs/model/settings#14-value-display')}> scaling options </Link>  to your preference. Value Display is set to full year and
scaling at 0.0m by default. Click on '^' option to collapse this property.
</div> <br />

<div style={{textAlign: 'center'}}>
  <Zoom>
<img alt="ValQ logo" src={useBaseUrl('/doc-images/Simulation/sim10.png')} />
  </Zoom>

## 5. Key Inputs

<div style={{textAlign: 'justify'}}>
Any important node that you consider as Key Input can be added as a Key Input to view it quickly instead of searching for it
in the model. The Key Input is displayed along with it's value and variance percentage when simulated. You can go to <Link to={useBaseUrl('docs/model/settings#15-key-inputs')}> settings </Link> 
to add/remove Key Inputs. Click on '^' option to collapse this property.
</div> <br />

<div style={{textAlign: 'justify'}}>
In the below example, we have added two Key Inputs and the node 'Chris Morris' gets highlighted

## 6. KPIs

<div style={{textAlign: 'justify'}}>
Any important node that you consider as Key Performance Indicator can be added as a KPI to view it quickly instead of searching for it
in the model. The KPI is displayed along with it's value and variance percentage when simulated. You can go to <Link to={useBaseUrl('docs/model/settings#16-kpis')}> settings </Link> 
to add/remove KPIs. Click on '^' option to collapse this property.
</div> <br />

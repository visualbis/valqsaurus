There are two ways to switch to the Table View in the Simulate screen:

1. By clicking on the table icon at the top of the Simulate tab canvas. 
2. By clicking on the Table option under the Visualization section of Navigation Panel.

## 2. Perform simulation

<div style={{textAlign: 'justify'}}>
Hover over the trend line on a node to perform simulation. The simulation results for each node with percentage change are available
on top of the canvas.  
</div> <br />

<div style={{textAlign: 'center'}}>
<Zoom>
<img alt="Settings" src={useBaseUrl('/doc-images/simulate_tab/table_view/tvv-19.png')} />
  </Zoom>
</div> <br />

Click on Edit icon to manually change the values.

<div style={{textAlign: 'center'}}>
<Zoom>
<img alt="Settings" src={useBaseUrl('/doc-images/simulate_tab/table_view/tvv-20.png')} />
  </Zoom>
</div> <br />

<div style={{textAlign: 'justify'}}>
Use the bin icon to remove a simulation and remove all the simulation at once, click on Reset all option.
</div>

## 6. Add Comments

<div style={{textAlign: 'justify'}}>
Click on the comment icon to open the Comments section. Comments can be added/edited/deleted to the model at node granularity.
This feature is same <Link to={useBaseUrl('docs/simulation/simulation_canvas#17-add-comments')}>Add Commets</Link> with Tree View.

## 9. Customize columns

<div style={{textAlign: 'justify'}}>
By clicking on settings, you can see the following screen.
On the left, you can search for columns, select or unselect them based on whether you want it to be displayed.
On the right, you can reorder columns by drag and drop. Click on Reset to default to erase the changes and go back to default.

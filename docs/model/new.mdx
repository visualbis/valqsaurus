## 2. Create a Simple Model

<div style={{textAlign: 'justify'}}>
To build a Simple Model, first you need to import data into Power BI and activate ValQ. This is followed by creating a
simple model on ValQ.
</ div> <br />

<div style={{textAlign: 'justify'}}>
The steps have been covered under <Link to={useBaseUrl('docs/intro/model')}> Using ValQ - An Example </Link> 
or you can refer <a href="https://valq.com/wp-content/uploads/valq-for-microsoft-power-bi-beginners-tutorial.pdf" target="_blank"> this pdf </a> 
 for step by step instructions on how to build a Simple Model. <br />
</ div> <br />

#### Edit a Simple Model

<div style={{textAlign: 'justify'}}>
The nodes on a Simple Model are disabled and you cannot perform any configuration on it. 
But you can generate a copy of the Parent node and proceed to configure it based on your needs.
</ div> <br />

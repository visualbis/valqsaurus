Create New Model

The ValQ model can be created in Power BI using any of the below-listed options by clicking on the Model tab.


ModelDescription
1Open a Sample ModelTo view a sample ValQ model
2Create a Simple ModelValQ builds a simple model for you from the data you import
3Create an Advanced ModelYou get to build a ValQ model from scratch from the data you import
4Import an Advanced ModelAdvanced users can import an already built existing model

Before starting to build a ValQ model, the first time users can click on this icon to learn more about ValQ and watch tutorials.

#1. Open a Sample Model

By clicking the Open a Sample Model option, you will be able to view the list of available samples.


For our example, the sample “Mining Industry” has been selected.


The Sample Mining Industry Model has been successfully imported.

#2. Create a Simple Model

To build a Simple Model first, you need to import data into Power BI. This step is followed by creating a simple model on ValQ.

The steps have been already covered under Using ValQ - An Example section. Or you can refer this pdf for step by step instructions on how to get started to build a Simple Model.

#Edit a Simple Model

When a node is selected, you will be able to only view the model with Parent and Children nodes and you cannot perform any configuration on it. But you can generate a copy of the Parent node and proceed with Configuration based on your choice (see Figure below).

#Dynamic Scaling

Based on your data source, the model will get created and the Value Display (scaling) for all the nodes will be updated appropriately based on the data source (see Figure below). For our example, the Number Scaling is “0m”.

#Sum of periods

The time aggregation of the Parent node is always set to Sum of Periods of Child nodes on a Simple Model.

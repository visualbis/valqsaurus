Model Tab

In the Model tab you can visualize your data and build models.

note

Switch from Power BI's Reading view to Edit view to use ValQ's Model tab.

Model Screen

The Model screen has 2 sections; namely:

  • Nodes Panel: This is the sidebar-panel on the left, to edit the model nodes and build the model.
  • Canvas(visual): This is the canvas on the right, to visualize the model in tree/table structure.

The Model tab has five other tabs as shown in the screenshot below:

New#

To start creating new models select one of the four options as shown in the screenshot below:

Select a Sample Model

Node#

In the Node tab you can configure the design and functionality of the Nodes of a model.

Nodes configuration panel

Data series#

Data Series tab allow you to configure the data settings such as Data Series Manager, Time Aggregation Labels and Data Sorting.

Data Series configurations

Settings#

In the Settings tab you can configure the parameters such as Navigational Panel, Nodes, Canvas, Number Formatting, Conditional Formatting, Color Theme and PopUp screen.

Settings nav-panel

General#

In the General tab you can configure the general settings such as License Registration and Export your model.

General options

In the upcoming sections, you will learn to build a ValQ model using these options in detail.

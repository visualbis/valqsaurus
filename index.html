Introduction to ValQ

ValQ helps businesses run simulations, plans & analysis in a very visual and intuitive manner.

What can I use ValQ for?#

ValQ is a powerful, multi-purpose tool and can be used across all functions in an organization. ValQ helps decision-makers and executives simulate various scenarios on-the-fly and analyze outcomes to arrive at actionable decisions. ValQ supports use cases such as what-if simulations, scenario comparisons, planning, forecasting, variance analysis, value driver analysis, and more.

ValQ use-cases

ValQ Collaborate features include:

  • Scenario collaborations
  • Comments and Annotations
  • Planning Series Collaboration
  • Audit Log
  • MemSQL DB Storage
  • Web Data Source
  • Private Tenant
  • Writeback API
  • Security

For more info, check out ValQ Pricing page.

Who can use ValQ?#

ValQ is for everyone ranging from C-suite executives to analysts across departments and functions.

ValQ extends its functionality beyond financial & operations planning.

For example,

  1. A customer service rep can leverage ValQ to simulate headcount needs based on estimated growth in call volume, upcoming product launches, marketing campaigns, and more.
  2. A marketing manager can leverage ValQ to analyze web traffic, online assets, conversion rates, etc. to calculate marketing ROI.

ValQ domain expertise

What do I need to run ValQ?#

ValQ is an independent solution that does not require any infrastructure, database, service, or IT support.

All you need is Microsoft Power BI to run ValQ.

Microsoft Power BI desktop is a free software that you can download here.

With ValQ available as a standalone visual for Microsoft Power BI in the AppSource, you can get running in a matter of minutes. Learn how to download & install ValQ here.
